# VMware and Kubernetes Home Lab Portfolio

This repository documents a home lab environment built to demonstrate practical administration skills across VMware vCenter, vSAN, NSX, and Kubernetes.

All work is performed in a lab environment. This repo is proof of work via repeatable runbooks, scripts, diagrams, and troubleshooting notes.

## What This Repo Demonstrates
- vCenter administration: cluster configuration, monitoring, RBAC, alarms, lifecycle tasks
- vSAN: storage policies, health checks, troubleshooting workflows
- NSX: logical networking and segmentation concepts, baseline security policy, validation testing
- Kubernetes: cluster bootstrap, workload deployments, operational runbooks

## Repo Map
- Runbooks: `runbooks/`
- Architecture and standards: `docs/`
- Troubleshooting: `troubleshooting/`
- Evidence (sanitized): `evidence/`

## Evidence and Safety
No secrets, license keys, tokens, or personal identifiers are committed.
All screenshots are sanitized.
